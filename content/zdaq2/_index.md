Mymind 终于全面推送 API 和 MCP 功能了，可以很方便地接入 Agent 或 ChatGPT 和 Claude。使用 API 接入 Agent 可以与各类软件结合建立更顺畅的保存流程，即便暂时没有折腾的需求，使用 MCP 也可以弥补官方在非英文内容自动 Tag 不够完美、偶尔搜索不够细致的问题。

上个月仔细看了订阅权限后，我把订阅方案从 Mastermind 降级到了 Student of Life，其实已经能够满足我的所有需求了，每个月还能省几美元。
